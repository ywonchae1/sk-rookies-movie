# 영화 조회 서비스 제작

> [!TIP]
> SK Rookies 지능형 애플리케이션 개발자 양성과정 과제입니다.

## FEATURE

- [THE MOVIE DB](https://api.themoviedb.org) API를 활용한 영화 제목 검색
- 영화 포스터, 시놉시스, 동영상 클립 상세 내용 제공
- [react-youtube](https://www.npmjs.com/package/react-youtube) 라이브러리를 활용한 유튜브 영상 재생
